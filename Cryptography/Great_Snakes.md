# Great Snakes

## Step by Step Solution
1. Read the challenge — it gives you a short Python 3 snippet and tells you to run it.
2. CryptoHack provides an in-browser Python execution environment for this challenge — paste the given code into it.
3. Run the code. It will print the flag directly to the output.
4. Copy the flag exactly as printed and submit it.

## Concept
Confirms you can execute Python (the primary language used throughout CryptoHack) and introduces the platform's built-in code runner.
