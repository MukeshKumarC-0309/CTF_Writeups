# Warmed Up

## Step by Step Solution
1. Read the challenge — it gives you a number in hexadecimal (base 16, written like `0x3D`) and asks for its decimal (base 10) or ASCII value.
2. Open a terminal.
3. Type `python3` to open the Python interpreter.
4. Type `print(0x3D)` (replace `3D` with whatever number the challenge gave you). Python instantly converts it for you.
5. Take the answer it prints out and wrap it in the flag format: `picoCTF{answer}`.
6. Submit that as your flag.
